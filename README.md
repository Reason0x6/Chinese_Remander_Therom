# Chinese_Remander_Therom
-------------
A Java Program to take a Chinese Remander Therom problem and output worked answers

## Input
Once the program is loaded it will ask you for N & A

This comes from Ax mod N = 1

e.g 7x mod 40 = 1

N = 40

A = 7
